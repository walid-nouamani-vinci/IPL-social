## Description
Le projet IPL Social implémente une fonction de validation d’adresse email en TypeScript.
Le développement suit la méthode TDD (Test Driven Development) avec des tests unitaires écrits avant le code.
Une pipeline CI via GitHub Actions permet d’exécuter automatiquement les tests à chaque push.
